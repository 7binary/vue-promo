> ROOT - Корневое хранилище

```
loading – признак загрузки формы, до ответа сервера равен "true"
profile_id – идентификатор авторизованного участника
token – JWT-токен, полученный при авторизации
user – массив с данными текущего участника (ФИО, email, баланс кошелька)
settings - массив с настройками проекта
HandleError() – универсальный обработчик ошибок запросов axios (catch)
GetProfile() – получение (обновление) данных по текущего участнику
getters.ax – получение экземпляра axios для post-запросов
getters.axl – получение расширенного экземпляра axios для post-запросов с поддержкой loading
getters.isAuthenticated – авторизован ли участник
```

> SETTINGS - настройки проекта
```
taxes - требовать ли паспорт с участника на проекте
extranet - закрывать ли логин/регистрацию/меню-гостя (участники логинятся по ссылке)
shop - отображать ли модуль магазина товаров
payments - отображать ли модуль платежей
courses - включен ли модуль обучения
sales - включен ли модуль продаж
news - включен ли модуль новостей
instructions - включен ли модуль презентаций/интсрукций
eps - отображать ли модуль сертификатов ЭПС
eps_confirmed - заказанные сертификаты ЭПС требуют подтверждения модератором
registration - тип регистрации участника (по спискам / свободная регистрация)
layout - шаблон сайта (меня сверху / меню слева)
```

> EPS - Хранилище сертификатов, корзины, заказов. Модуль: modules/eps

```
cards – сертификаты, доступные для заказа на сайте
orders – заказы сертификатов авторизованного участника
cart – корзина с выбранными участником сертификатами
cartLength – сколько всего позиций в корзине
cartSummary – на какую сумму всего позиций в корзине
GetOrders() – получение списка всех заказов текущего участника
AddToCart() – добавление сертификата в корзину
ChangeQty() – изменение количества конкретного сертификата в корзине
RemoveCard() – удаление сертификата из корзины
CountCart() – пересчет (обновление) cartLength и cartSummary
ClearCart() – очистка всех позиций корзины
getters.cartItems – получение списка корзины для отправки на сервер по API
```
